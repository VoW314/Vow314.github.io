**Status: Active Development** · [Live App](https://kdza4xp8cqv2e6wsnhz76c.streamlit.app/)

A Guild vs Guild match analytics platform for the game *Where Winds Meet*. Uses post-game data and object detection from match recordings to analyze individual player performance during competitive encounters.

- Applies **computer vision / object detection** to extract structured data from gameplay video
- Aggregates post-match statistics and models player performance over time
- Integrates **ChatGPT mini** to auto-generate coaching tips based on each player's analytics
- Hosted publicly on **Streamlit**

**Stack:** Python · Object Detection · ChatGPT API · Streamlit · Pandas
