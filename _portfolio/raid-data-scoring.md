Collaborated with esports coach **Richard Staats** to build a data-driven performance improvement system for a competitive guild's PvE (raid) encounters.

- Collected time-series performance data from guild members after each PvE encounter and team fight
- Implemented an **ELO-based scoring algorithm** per raid that accounts for raid difficulty and class composition
- Tracked player grades over time to identify underperformers and target coaching interventions
- Used insights to directly advise players on build optimization and skill rotations
- **Result:** Raised the guild's average damage and healing output as bottom-tier players improved measurably

**Stack:** Python · Pandas · ELO Rating · Time-Series Analysis
