Collaborated in a team of 5 students to build a cloud-based healthcare management application on **AWS EC2**.

- Designed a secure account creation and medical file storage drive backed by SQL
- Engineered a **ChatGPT 4o** medical triage assistant to give non-diagnostic health information to users
- Integrated **NPPES** and **FDA Drug API** searches so users can find, save, and share provider and medication information

**Stack:** Java · WildFly · AWS EC2 · MySQL · ChatGPT API · NPPES API · FDA API
